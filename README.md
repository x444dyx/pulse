# Pulse

A minimal, precision-timing arcade game built with React and Vite.

------------------------------------------------------------------------

## Tech Stack

-   React 19
-   Vite
-   TypeScript
-   TailwindCSS
-   Motion

------------------------------------------------------------------------

## Development

### Install dependencies

``` bash
npm install
```

### Start development server

``` bash
npm run dev
```

### Build for production

``` bash
npm run build
```

### Preview production build

``` bash
npm run preview
```

------------------------------------------------------------------------

## Deployment

Pulse is a static frontend application and can be deployed to:

-   Vercel
-   Netlify
-   GitHub Pages
-   Any static hosting provider

No backend or API keys required.
